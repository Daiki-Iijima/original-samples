## このリポジトリについて

シンプルな構造を意識して、学習目的や他アプリの実装にそのまま使えるような機能を組み込んだものなど、雑多なプロジェクトをまとめたリポジトリです。

## 各種プロジェクト

プロジェクトごとに説明と動作サンプルが「README.md」に記載されています。 

ここではざっくりとした説明だけになります。

- Swift(iOS)
  - [AVFoundationVideoCaptureSample](https://github.com/Daiki-Iijima/original-samples/tree/main/AVFoundationVideoCaptureSample) : シンプルなカメラ映像周りのハンドリングを検証
  - [DrawingPlayground](https://github.com/Daiki-Iijima/original-samples/tree/main/DrawingPlayground) : お絵かきアプリ
  - [DtoPlayground](https://github.com/Daiki-Iijima/original-samples/tree/main/DtoPlayground) : DTOという考え方をシンプルなアプリで検証
  - [ZoomPanPlayground](https://github.com/Daiki-Iijima/original-samples/tree/main/ZoomPanPlayground) : シンプルなズームイン・ズームアウト・パン機能
  - [ZoomPanPlaygroundEx](https://github.com/Daiki-Iijima/original-samples/tree/main/ZoomPanPlaygroundEx) : ZoomPanPlaygroundに拡張機能を追加
