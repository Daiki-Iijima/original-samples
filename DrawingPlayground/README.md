## このプロジェクトについて

このプロジェクトは、「お絵かきアプリ」をできるだけ整理してシンプルに実装している

また、お絵かき機能については、DrawKitというパッケージにまとめて開発しているので、他のアプリでも組み込みやすいような形式にしてある(これでいいのかは疑問が残るが)

主要機能については以下

- モード切替機能
- ペン書き込み機能
    - 色変更
    - 太さ変更
    - 透明度変更
- スタンプ機能
    - 色変更
    - 太さ変更
    - 透明度変更
- 消しゴム機能
    - 太さ変更
- Redo・Undo機能
- 一括クリア機能
- 描画結果の書き出し機能
